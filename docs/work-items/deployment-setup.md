# Work Item: GitHub/GitLab Pages Deployment

## Status
Planned

## Outcome
CI/CD pipelines for deploying the static site to GitHub Pages first, then GitLab Pages.

## Main Quests
1. Create GitHub Action for automated build and deploy.
2. Create GitLab CI configuration for the same purpose.
3. Validate that the same generator output works on both platforms without modification.

## Acceptance Criteria
- Site is live on GitHub Pages.
- Site is live on GitLab Pages.
- The build process is identical for both.
